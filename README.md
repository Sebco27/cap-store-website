# Cap Store Website

A modern product showcase website for a cap store built with **React**, **TypeScript**, and **Vite**. The project is intended to present products and store information. It does not include online purchasing functionality.

## Tech Stack

* React
* TypeScript
* Vite
* ESLint

## Features

* Responsive design
* Modern UI
* Place orders
